# Nyaa Watcher

A Windows desktop torrent browser for [nyaa.si](https://nyaa.si), built with Rust and Slint.


## Build

**Requirements**
- Rust (stable)
- Windows 10/11

```bash
git clone https://github.com/nora-2222/nyaa-watcher.git
cd nyaa-watcher

# Debug
cargo run

# Release
cargo build --release
# target/release/nyaa-watcher.exe
```
